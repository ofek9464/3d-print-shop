ℹ️ Security Notes

⚠️ Development Mode

This admin panel uses client-side authentication for development purposes. Before going to production, you need to implement:

  • Server-side authentication with bcrypt password hashing
  • JWT tokens with secure HTTP-only cookies
  • HTTPS/SSL certificate
  • CSRF protection
  • Rate limiting on login attempts
  • Database-backed product/pricing storage
  • Stripe payment integration

See the project documentation for the full production deployment guide.
